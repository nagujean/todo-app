# Team Collaboration Feature - Test Checklist

## Overview
팀 협업 기능에 대한 수동 테스트 체크리스트입니다.

**테스트 환경**: https://todo-app-2-tan.vercel.app
**테스트 일자**: 2026-01-23
**버전**: c26604c

---

## 1. 팀 생성 (Create Team)

### 기본 기능
- [ ] 팀 전환기에서 "새 팀 만들기" 버튼 클릭 시 모달 열림
- [ ] 팀 이름 입력 필드 표시됨
- [ ] 팀 설명 입력 필드 표시됨 (선택)
- [ ] 팀 이름 없이 제출 시 오류 메시지 표시
- [ ] 팀 이름 입력 후 "만들기" 클릭 시 팀 생성됨
- [ ] 생성된 팀이 팀 전환기 목록에 표시됨
- [ ] 생성자가 자동으로 owner 역할 부여됨

### 유효성 검사
- [ ] 팀 이름 최대 100자 제한 동작
- [ ] 빈 공백만 입력 시 제출 불가
- [ ] 특수문자 포함 팀 이름 생성 가능

---

## 2. 팀 전환 (Team Switcher)

### 기본 기능
- [ ] 헤더에 팀 전환기 드롭다운 표시됨
- [ ] 기본값 "개인" 선택됨
- [ ] 드롭다운 클릭 시 팀 목록 표시
- [ ] 팀 선택 시 해당 팀으로 전환됨
- [ ] 전환 시 페이지 타이틀이 팀 이름으로 변경됨
- [ ] "개인" 선택 시 개인 할 일 목록으로 전환됨

### 상태 유지
- [ ] 페이지 새로고침 후에도 선택된 팀 유지됨
- [ ] 로그아웃 후 재로그인 시 팀 목록 정상 로드

---

## 3. 팀 멤버 관리 (Team Members)

### 멤버 목록 조회
- [ ] 사용자 메뉴에서 "팀 멤버" 클릭 시 모달 열림
- [ ] 현재 팀의 모든 멤버 목록 표시
- [ ] 각 멤버의 이름, 이메일, 역할 표시됨
- [ ] 현재 사용자에 "(나)" 표시됨
- [ ] 역할별 아이콘/색상 구분됨

### 역할 변경 (Admin 이상)
- [ ] Admin/Owner가 다른 멤버의 역할 드롭다운 클릭 가능
- [ ] 역할 변경 옵션: admin, editor, viewer
- [ ] Owner 역할은 변경 불가 (드롭다운 없음)
- [ ] 자기 자신의 역할 변경 불가
- [ ] 역할 변경 후 즉시 UI에 반영됨

### 멤버 제거 (Admin 이상)
- [ ] Admin/Owner가 멤버 제거 버튼 표시됨
- [ ] Owner는 제거 불가 (버튼 없음)
- [ ] 제거 클릭 시 확인 다이얼로그 표시
- [ ] 확인 후 멤버가 목록에서 제거됨

### 권한 검증
- [ ] Editor/Viewer는 역할 변경 드롭다운 없음
- [ ] Editor/Viewer는 멤버 제거 버튼 없음

---

## 4. 팀원 초대 (Invite)

### 이메일 초대
- [ ] 사용자 메뉴에서 "팀원 초대" 클릭 시 모달 열림
- [ ] "이메일" 탭이 기본 선택됨
- [ ] 이메일 입력 필드 표시됨
- [ ] 역할 선택 드롭다운 표시됨 (admin, editor, viewer)
- [ ] 유효하지 않은 이메일 형식 입력 시 오류
- [ ] 초대 전송 후 성공 메시지 표시

### 링크 초대
- [ ] "링크" 탭 클릭 시 링크 초대 UI 표시
- [ ] 역할 선택 가능
- [ ] "링크 생성" 클릭 시 초대 링크 생성됨
- [ ] 생성된 링크 표시됨
- [ ] "복사" 버튼 클릭 시 클립보드에 복사
- [ ] 복사 성공 피드백 표시

### 초대 권한
- [ ] Owner/Admin만 초대 가능
- [ ] Editor/Viewer는 초대 메뉴 접근 불가 (또는 비활성화)

---

## 5. 역할별 권한 테스트 (RBAC)

### Owner 권한
- [ ] 팀 설정 변경 가능
- [ ] 모든 멤버 역할 변경 가능 (자신 제외)
- [ ] 모든 멤버 제거 가능 (자신 제외)
- [ ] 팀원 초대 가능
- [ ] 할 일 생성/수정/삭제 가능
- [ ] 팀 삭제 가능

### Admin 권한
- [ ] Editor/Viewer 역할 변경 가능
- [ ] Owner/다른 Admin 역할 변경 불가
- [ ] Editor/Viewer 제거 가능
- [ ] 팀원 초대 가능
- [ ] 할 일 생성/수정/삭제 가능
- [ ] 팀 삭제 불가

### Editor 권한
- [ ] 멤버 역할 변경 불가
- [ ] 멤버 제거 불가
- [ ] 팀원 초대 불가
- [ ] 할 일 생성/수정/삭제 가능

### Viewer 권한
- [ ] 멤버 역할 변경 불가
- [ ] 멤버 제거 불가
- [ ] 팀원 초대 불가
- [ ] 할 일 조회만 가능 (생성/수정/삭제 불가)

---

## 6. 팀 나가기 (Leave Team)

- [ ] 팀 설정에서 "팀 나가기" 옵션 표시
- [ ] Owner는 팀 나가기 불가 (소유권 이전 필요)
- [ ] 나가기 클릭 시 확인 다이얼로그 표시
- [ ] 확인 후 팀 목록에서 해당 팀 제거됨
- [ ] 자동으로 "개인" 워크스페이스로 전환됨

---

## 7. 실시간 동기화

- [ ] 다른 기기에서 팀 멤버 추가 시 즉시 반영됨
- [ ] 역할 변경 시 해당 멤버에게 즉시 반영됨
- [ ] 멤버 제거 시 해당 멤버에게 즉시 반영됨
- [ ] 할 일 변경 시 모든 팀원에게 즉시 반영됨

---

## 8. 모바일 환경 테스트

### UI/UX
- [ ] 팀 전환기가 모바일 화면에 적절히 표시됨
- [ ] 드롭다운 메뉴가 터치로 정상 동작
- [ ] 모달 다이얼로그가 화면에 맞게 표시됨
- [ ] 입력 필드에 모바일 키보드 정상 표시
- [ ] 스크롤이 필요한 경우 정상 동작

### PWA
- [ ] 홈 화면에 추가 가능
- [ ] 앱으로 실행 시 팀 기능 정상 동작
- [ ] 오프라인에서 캐시된 데이터 표시

### 반응형
- [ ] 가로/세로 모드 전환 시 레이아웃 정상
- [ ] 다양한 화면 크기에서 UI 깨짐 없음

---

## 9. 에러 처리

### 네트워크 오류
- [ ] 오프라인 상태에서 적절한 오류 메시지 표시
- [ ] 네트워크 복구 후 자동 재연결

### 권한 오류
- [ ] 권한 없는 작업 시도 시 오류 메시지 표시
- [ ] Firestore 보안 규칙에 의해 차단됨

### 입력 오류
- [ ] 잘못된 입력에 대한 유효성 검사 메시지 표시
- [ ] 중복 초대 방지

---

## 10. 엣지 케이스

- [ ] 팀 이름이 매우 긴 경우 (100자) UI 표시
- [ ] 멤버가 많은 경우 (10명+) 목록 스크롤
- [ ] 동시에 여러 사용자가 같은 멤버 역할 변경 시도
- [ ] 초대 링크 만료 후 접근 시도
- [ ] 이미 팀에 있는 사용자 재초대 시도
- [ ] 자기 자신을 초대 시도
